# TranslatedPySimpleGUI

An example Python3 App with PySimpleGui and Babel

## How to install

1. Create and load venv
2. Install dependencies <br>
   `pip install -r requirments.txt`

2. Create translations <br>
   `./create_translations.bat`

## How to run

1. Run<br>
   `python main.py`
